# Movie Favorites List

A simple React application to manage your favorite movies. Built as part of **Lesson 9** to demonstrate the `.map()` method, state management, and dynamic list rendering.

## ✨ Features

- Add new movies to the list
- Remove movies from the list
- Responsive and clean UI
- Uses React `useState` hook for state management

## 🛠 Technologies Used

- React.js
- JavaScript (ES6+)
- CSS (via App.css)
